# Zen-Self-Ref.md — 禅宗自指结构类型库

本文件为 chanzong 技能的步骤二提供**五种自指类型的完整定义、识别信号、奇异循环图模板和经典案例库**。

---

## 类型一：火求火型

### 公式

```
学人以为自己缺少X → 学人向师父"求X"
→ 师父："把X拿来给我看"
→ 学人寻找X → 发现"X不可得"
→ 觉悟：原来能"求"的本身就是X
```

### 识别信号（关键词）

觅、求、乞、将X来、与汝X、不可得、竟、已与汝X竟、安心竟、忏罪竟

### 奇异循环图模板

```
[现象层] 学人体验到"不安"/"有罪"/"被缚"/"缺什么"
    ↓ 以为自己缺少X，向外求
[认知层] 学人把X当成一个可以被"获得"的对象
    ↓ 师父："把X拿来"
[行动层] 学人试图把"心"/"罪"/"缚"当成一个物来寻找
    ↓ 发现X不是一个"物"——不可得
[拐点] "觅心了不可得" / "觅罪不可得"
    ↓ 寻找的动作本身崩塌
[崩溃] "与汝安心竟" / "与汝忏罪竟"
    ↓
[觉悟] 能"不安"的就是"安心"——二者本来是一个
```

### 经典案例库

| 案例 | 学人 | 师父 | X = ? | 拐点句 | 崩溃句 |
|------|------|------|-------|--------|--------|
| 慧可安心 | 慧可 | 达摩 | 安心 | "觅心了不可得" | "与汝安心竟" |
| 僧璨忏悔 | 僧璨 | 慧可 | 忏悔 | "觅罪不可得" | "与汝忏罪竟" |
| 道信求解脱 | 道信 | 僧璨 | 解脱 | "无人缚" | "何更求解脱乎" |
| 丙丁童子来求火 | 丙丁童子 | 法眼 | 火 | （被法眼反问"谁是丙丁"） | （童子自省，自己就是火） |

### 变体：师父是"把火行"而不自知

当师父（或者求法者）自己说出了自指隐喻（如桂琛说"大师是把火行"），但完全不知道这句话在说他自己——这仍然是火求火型，但拐点尚未被学人自己认出。此时：学人=正在把火行而不自知的那个人。

## 类型二：二元互依型

### 公式

```
学人用二元对立的框架提问（A还是非A？）
→ 师父指出A和非A互相定义对方
→ "A即非A"
→ 二元框架本身崩溃
→ 觉悟：原来"两个"是假象，"一个"才是实相
```

### 识别信号（关键词）

明/暗、生/死、有/无、与么/不与么、是/不是、同/不同、双明亦双暗、同生亦同死

### 奇异循环图模板

```
[框架层] 学人以"A还是非A"的二元框架提问
    ↓ 预设A和非A是互斥的两个东西
[揭示层] 师父指出"双明亦双暗"——A和非A互相依存
    ↓ 如同一枚硬币的正面和反面
[崩溃层] 学人的二元框架无法处理"A即非A"这个命题
    ↓ 框架崩塌——"如牛无角""如虎戴角"
[觉悟] 没有A，也没有非A——只有"不落有无"的实相
```

### 经典案例库

| 案例 | A | 非A | 师父的关键操作 | 崩溃隐喻 |
|------|---|-----|--------------|---------|
| 双明亦双暗 | 明 | 暗 | "双明亦双暗"→"同生亦同死" | "如牛无角""如虎戴角" |
| 岩头"与么与么，不与么不与么" | 与么（这样） | 不与么（不这样） | 对"这样"的肯定和否定同时进行 | 桂琛无法理解→三天后才追问→"同生亦同死" |
| 慧能"不思善不思恶" | 善 | 恶 | 暂停所有二元判断 | "哪个是明上座本来面目" |

### 变体：云门一字关 / 临济喝 / 德山棒

当师父用一个非二元的动作（一字、一喝、一棒）来截断学人的二元思维时——"喝"既不是"肯定"也不是"否定"——它不在二元的任何一边。

## 类型三：自我递归型

### 公式

```
学人向外寻求"我是谁"的答案
→ 师父让学人看到：你所寻求的"答案"处处都是"你"的显现
→ 每一个"非我"都是"我"的证明
→ 但"我"不能被任何单一的"非我"所穷尽
→ 觉悟："我"和"非我"不是两个——前者通过后者认识自己
```

### 识别信号（关键词）

渠、影、镜、谁、哪个、本来面目、主人公、是我是渠、渠今正是我、我今不是渠

### 奇异循环图模板

```
[现象层] "我"在寻找"我是谁"
    ↓ 向外看——看到水中倒影、听到呼唤、遇到他者
[反射层] 倒影/呼唤/他者——它们都是"我"的显现
    ↓ "渠今正是我"——影就是我
[超越层] 但"我"不只是影——"我今不是渠"
    ↓ "我"是能产生影的那个——影不能穷尽"我"
[循环] "我"通过不断产生"非我"来认识自己
    ↓ 每一次向外看，都是在看自己
[觉悟] "切忌从他觅，迢迢与我疏"
```

### 经典案例库

| 案例 | "我"的形式 | "非我"的形式 | 关键偈/语句 |
|------|-----------|-------------|-----------|
| 洞山睹影 | 洞山本人（能见的） | 水中倒影（所见的） | "渠今正是我，我今不是渠" |
| 慧能度慧明 | 本来面目（觉性） | "不思善不思恶"的间隙 | "哪个是明上座本来面目" |
| 罗山召桂琛 | 桂琛的觉性 | 罗山的呼唤 + 桂琛的回应 | "山召师，师应诺。山云：双明亦双暗。" |

## 类型四：语言自毁型

### 公式

```
师父说出一句话 / 做出一个动作
→ 这句话/动作的功能不是传达信息——而是销毁自己
→ 学人如果试图"理解"它——它就不起作用
→ 学人如果在它面前停下来——话已经不需要了
```

### 识别信号（关键词）

喝、棒、默然、良久、一字答、转身、不回答、答非所问、问东答西

### 奇异循环图模板

```
[语言层面] 师父说了一个字 / 做了一个动作
    ↓ 学人试图在语言层面理解它
[断裂层面] 这个字/动作破坏了"问题-答案"的正常对应
    ↓ 学人的思维在这里卡住了
[拐点] 如果学人继续追问——进入死循环
      如果学人停下——语言已经自毁，不需要再"理解"了
[觉悟] 不是"懂了那个字"——是"不再需要那个字了"
```

### 经典案例库

| 案例 | 语言自毁形式 | 学人的反应 | 是否"成功" |
|------|------------|-----------|----------|
| 云门一字关 | "祖""是""普"——一字答 | 追问=死，契入=活 | 取决于学人 |
| 赵州"吃茶去" | 不论问什么都一句"吃茶去" | 院主追问→被同一个字打回 | 院主未悟 |
| 临济喝 | 一喝 | 非追问非不追问——直接对应 | 如果是主→悟 |
| 德山棒 | 一棒 | 打掉学人所有的"求" | 如果是真求者→悟 |
| 罗山召桂琛 | "山召师，师应诺"——呼唤-回应 | 桂琛未悟，三天后追问 | 未成功 |
| 南泉斩猫 | 一刀——用死亡本身作为"语言" | 赵州脱鞋顶头上→活；其余众僧→死 | 只有赵州活 |

### 关键判断

**语言自毁型 ≠ 不说话。** 沉默可能是"玄中玄"（真的），也可能是"偏中正"（躲进空寂）。判断标准：
- 如果沉默是为了制造一个学人必须跨越的语言断裂——真
- 如果沉默是师父自己的回避——假

## 类型五：循环引证型

### 公式

```
学人向师父提出一个论点/问题/质疑
→ 师父用学人自己的逻辑框架来反问学人
→ 学人发现自己无法在自己的逻辑框架内回答这个问题
→ 学人的逻辑框架自己推翻了自己
```

### 识别信号（关键词）

磨砖、把火行、汝既不会、却来问我、汝、尔、你（第二人称反问）

### 奇异循环图模板

```
[论点层] 学人："我认为X可以导致Y"
    ↓ 学人自己的逻辑前提：X类操作→Y类结果
[反问层] 师父用一个类比把学人的逻辑扩展
    ："如果X可以导致Y，那么X'可以导致Y'吗？"
    ↓ 学人自己的逻辑必然导致荒谬结论
[卡住] 学人："不可以。"
    ↓ 学人发现了自己逻辑的矛盾——但矛盾不在师父的类比里，
      在自己的前提里
[拐点] "磨砖既不成镜，坐禅岂能成佛？"
    ↓ 学人无法维护自己的前提而不陷入自相矛盾
[崩溃→觉悟] 放弃前提——不是"找到更好的X"，而是"X→Y的框架本身不成立"
```

### 经典案例库

| 案例 | 学人的前提 | 师父的循环引证 | 拐点句 |
|------|----------|-------------|--------|
| 怀让磨砖 | 坐禅→成佛 | 磨砖→成镜（荒谬）→∴坐禅→成佛（同样荒谬） | "磨砖既不成镜，坐禅岂能成佛？" |
| 桂琛"把火行" | 师父应该直接给我答案 | "大师是把火行" → 你手里有火（觉知），却在抱怨暗 | 桂琛说出"把火行"而不自知——拐点潜在他自己的话里 |
| 洞山对云岩 | "无情说法"——无情之物也能说法？ | 洞山：你听到了吗？云岩：我听到——那谁没听到？ | 洞山自己的"听"已经是"说法"的证明 |

### 变体：自发性循环引证

当学人**自己说出了自指隐喻而不自知**（桂琛"大师是把火行"）——他在循环引证自己。他批评师父的逻辑框架，恰恰暴露了他自己的盲点。此时：不需要师父再引证——学人的话本身就是回旋镖。

## 多类型叠加

一个公案经常同时包含多种自指类型。例如：

- **桂琛问罗山"双明亦双暗"** = 类型二（二元互依型）+ 类型四（语言自毁型：召→应诺）+ 类型五（循环引证："把火行"）
- **洞山睹影** = 类型三（自我递归型：渠/我）+ 类型四（语言自毁型：过水睹影是非语言的"触发"）
- **慧可安心** = 类型一（火求火型：觅心了不可得）+ 类型三（自我递归型：能觅的就是所觅的）

**规则**：检测出一个公案中**所有**匹配的自指类型，但只对**最核心的1-2个**做完整的奇异循环图。其余类型一句话标注即可。
